# Video-Game Library

This is the third project for the Flatiron School Web Development Course. This simple app lets you keep track of your videogame consoles along with their games. 

## Download Instructions

1. Clone from Github
2. Bundle Install 
3. Rails db:migrate 
4. Run rails server.
